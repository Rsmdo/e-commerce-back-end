  ## Installation
  Install npm , Mysql2, Sequalize, and dotenv(you can use your own package to hide your personal information)

  ## Usage
  Run mysql in the terminal and then run the schema file, after this exit mysql and run the seed (npm run seed) then start the server (npm start). Use insomia to test the routes.
